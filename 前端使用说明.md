# 前端使用说明

## 启动方式
由于前端是纯静态HTML/CSS/JS应用，**不需要启动任何前端服务**。您可以：

1. **直接在浏览器中打开**
   - 找到文件：`frontend/index.html`
   - 双击打开，或右键选择浏览器打开

2. **使用本地HTTP服务器（可选）**
   如果需要通过HTTP协议访问（推荐），可以使用以下方法：
   ```bash
   # 使用Python 3
   cd frontend
   python -m http.server 3000
   
   # 然后在浏览器访问：http://localhost:3000
   ```

## 测试步骤

1. **确保后端服务正在运行**
   - 后端服务已启动在：http://localhost:3001
   - 登录接口：http://localhost:3001/api/auth/login

2. **登录系统**
   - 打开前端页面（`index.html`）
   - 点击右上角"登录"按钮
   - 使用默认测试账号：
     - 管理员：`000` / `123456`
     - 教师：`101` / `123456`
     - 学员：`201` / `123456`

3. **测试学员列表功能**
   - 登录成功后，在左侧菜单点击"学员管理"
   - 查看学员列表，确认是否显示课程信息
   - 检查是否还有"500获取学员列表失败"错误

## 注意事项

- 前端已配置为自动连接到 `http://localhost:3001` 的后端服务
- 如果后端服务端口有变化，需要修改 `frontend/static/js/api.js` 中的 API_BASE_URL
- 确保浏览器没有阻止跨域请求（前端和后端不在同一端口时可能需要注意）
